## Client side
Originally, this project was generated by using `create-react-app` from React CLI with `yarn eject` to allow the developer to modify webpack and configuration.

## Step to start project
1. Execute `yarn` to install the dependencies.
2. Execute `yarn start` to start Dev Server with Hot reload module.

## Test
This project is using [Jest](https://jestjs.io/) for testing framework.
1. Execute `yarn test` to start testing.
